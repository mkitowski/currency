Online currencies exchange service


## Technologies: 
- React
- ES6
- API
- React-creat-app
- styled-component
- recharts
- material-designs

## Main functionality:

- User login
- Simulation of live updated currency rates
- Online currency converse
- Display user exchange history
- Display currency rates history on line chart
- Customised Landing Page
